# 販売管理システム - 売上入力画面

Vue.js 3 で作成した販売管理システムのデモ用売上入力画面です。

## 機能

- **ヘッダー情報入力**: 日付、得意先、担当者
- **明細入力**: 商品、個数、単価（複数行対応）
- **自動計算**: 金額、小計、消費税(10%)、合計金額
- **オートコンプリート**: 得意先、担当者、商品の検索機能
- **行操作**: 明細行の追加・削除

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` にアクセスしてください。

### ビルド

```bash
npm run build
```

## 技術スタック

- Vue.js 3 (Composition API)
- Vite
- 純粋な HTML/CSS（UIライブラリなし）

## 注意事項

このアプリケーションはデモ用です。入力されたデータは保存されません。
